# <Await>

用于通过自动错误处理呈现延迟值。 (opens new window)请务必查看延迟数据指南 (opens new window)，因为有一些 API 可与此组件一起使用。

import { Await, useLoaderData } from "react-router-dom";

function Book() {
  const { book, reviews } = useLoaderData();
  return (
    <div>
      <h1>{book.title}</h1>
      <p>{book.description}</p>
      <React.Suspense fallback={<ReviewsSkeleton />}>
        <Await
          resolve={reviews}
          errorElement={
            <div>Could not load reviews 😬</div>
          }
          children={(resolvedReviews) => (
            <Reviews items={resolvedReviews} />
          )}
        />
      </React.Suspense>
    </div>
  );
}

注意： 期望在 a或parent<Await>内部呈现以启用后备 UI。<React.Suspense>``<React.SuspenseList>

# 类型声明

declare function Await(
  props: AwaitProps
): React.ReactElement;

interface AwaitProps {
  children: React.ReactNode | AwaitResolveRenderFunction;
  errorElement?: React.ReactNode;
  resolve: TrackedPromise | any;
}

interface AwaitResolveRenderFunction {
  (data: Awaited<any>): React.ReactElement;
}

# children

可以是 React 元素或函数。

使用函数时，该值作为唯一参数提供。

<Await resolve={reviewsPromise}>
  {(resolvedReviews) => <Reviews items={resolvedReviews} />}
</Await>

使用 React 元素时，useAsyncValue (opens new window)将提供数据：

<Await resolve={reviewsPromise}>
  <Reviews />
</Await>;

function Reviews() {
  const resolvedReviews = useAsyncValue();
  return <div>{/* ... */}</div>;
}

# errorElement

当 promise 被拒绝时，error 元素代替子元素呈现。您可以使用访问错误useAsyncError (opens new window)。

如果 promise 被拒绝，您可以提供一个选项来通过挂钩errorElement在上下文 UI 中处理该错误。useAsyncError

<Await
  resolve={reviewsPromise}
  errorElement={<ReviewsError />}
>
  <Reviews />
</Await>;

function ReviewsError() {
  const error = useAsyncError();
  return <div>{error.message}</div>;
}

如果您不提供 errorElement，则被拒绝的值将冒泡到最近的路由级别errorElement (opens new window)，并可通过useRouteError (opens new window)挂钩访问。

# resolve

接受从延迟 (opens new window) 加载程序 (opens new window)值返回的承诺，以进行解析和呈现。

import {
  defer,
  Route,
  useLoaderData,
  Await,
} from "react-router-dom";

// given this route
<Route
  loader={async () => {
    let book = await getBook();
    let reviews = getReviews(); // not awaited
    return defer({
      book,
      reviews, // this is a promise
    });
  }}
  element={<Book />}
/>;

function Book() {
  const {
    book,
    reviews, // this is the same promise
  } = useLoaderData();
  return (
    <div>
      <h1>{book.title}</h1>
      <p>{book.description}</p>
      <React.Suspense fallback={<ReviewsSkeleton />}>
        <Await
          // and is the promise we pass to Await
          resolve={reviews}
        >
          <Reviews />
        </Await>
      </React.Suspense>
    </div>
  );
}
