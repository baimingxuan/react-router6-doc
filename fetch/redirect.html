redirect ​

由于可以在loaders和actions中返回或抛出响应，因此可以使用 redirect 重定向到另一个路由。

jsx
import { redirect } from "react-router-dom";

const loader = async () => {
  const user = await getUser();
  if (!user) {
    return redirect("/login");
  }
  return null;
};

这其实只是一个快捷方式：

jsx
new Response("", {
  status: 302,
  headers: {
    Location: someUrl,
  },
});

当重定向是为了响应数据时，建议在加载器和操作中使用 redirect ，而不是在组件中使用 useNavigate 。
